# Canvas


#### Features

* []() Choose any Color
* []() Choose any stroke width
* []() Background Fill
* []() Download your work
* []() Upload ur work from local PC and edit it
 
#### Installation 
1. Clone the repo
```sh
git clone https://github.com/vidhik2002/Canvas.git
```
2. Move into folder
```sh
cd Canvas
```
3. Run the commands
```sh
npm i
nodemon index.js
```
